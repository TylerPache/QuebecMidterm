Grading Page

Good Design Techniques – CRAP. Alt and title attributes for each image

Each image has an alt and title attribute. The color scheme could be more appealing but it provides a contrast so the text and images are more visible.

Organization of information – text and graphics make sense – logical – use the CSS float property for some images.

Each image is in accordance with the text it is paired with and is floated properly.

Content – interesting read, excellent grammar, perfect spelling. Please do not make the content of text on the inside pages short. Try to keep the first page (index) to one screen, no scroll bars.

I tried to keep the content brief and have more pictures than text. I can't guarantee it to be an interesting read but all spelling and grammar is correct with the correct accent marks on the letters.

Navigation – easy to get around, intuitive, well labeled

My navigation bar is present on each page, as well as having the title of the tab and a header on the page itself displaying which page you are on.

Use of HTML5 and CSS3 (both text and positioning) Valid HTML 5 and Valid CSS3 – Remember to use the validators demonstrated in class

All pages and CSS validate

You must have some JavaScript. The JavaScript must have a clear purpose and benefit your site.

I have JavaScript controlling the functionality of the survey as well as displaying the slide show on the home page and mouseover/mouseout events on the activities and points of interest page
